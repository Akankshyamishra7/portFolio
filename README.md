# Akankshya Mishra - Developer Portfolio

A modern, responsive portfolio website built with **React 19**, **TypeScript**, **Vite**, and **Tailwind CSS**. Showcasing projects, skills, education, and experience with a focus on performance and accessibility.

## 🎯 Features

- **Modern UI/UX**: Clean, professional design with dark theme and gradient accents
- **Responsive Design**: Fully mobile-friendly with adaptive layouts
- **Performance Optimized**: 
  - Lazy loading of components with Suspense
  - Code splitting via Vite
  - GPU acceleration for smooth animations
- **Accessibility First**:
  - Keyboard navigation support
  - Screen reader compatible
  - Focus management
  - Semantic HTML
- **Smooth Animations**: Mobile-optimized scroll animations
- **SEO Ready**: Semantic markup and meta tags

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Akankshyamishra7/portFolio.git
cd portFolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linting
npm lint
```
