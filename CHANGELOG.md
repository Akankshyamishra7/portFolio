# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-04-28

### Added
- Comprehensive README documentation with project structure and features
- Enhanced SEO meta tags in index.html (Open Graph, Twitter Cards)
- CHANGELOG file for version tracking
- Keywords and author meta tags for better discoverability
- Canonical URL for SEO optimization

### Improved
- Better documentation of tech stack and available scripts
- Enhanced project structure documentation
- Added feature highlights and accessibility information

## [1.0.0] - 2026-04-01

### Added
- Initial portfolio launch
- React 19 with TypeScript setup via Vite
- Responsive dark-themed design with Tailwind CSS
- Component lazy loading with Suspense for performance
- Mobile-optimized scroll animations
- Full accessibility support (keyboard navigation, ARIA labels, focus management)
- Custom hooks for accessibility and performance
- Mobile menu with focus trapping
- Smooth scrolling behavior with GPU acceleration
- Custom typography with multiple font families
- ESLint configuration for code quality

### Features
- Header with responsive navigation
- Profile summary section with highlights
- Education section
- Technical skills showcase
- Projects portfolio
- Hackathon/experience section
- Core competencies list
- Footer with contact information
- Accessibility announcer component
- Performance optimization utilities
